06 — Brand Voice & Usage
Do
✓ Use LOBB in all-caps in logos, marks, and headlines
✓ Use "Lobb" in sentence case for running UI copy and body text
✓ Lead with benefits — "Pay safely" before "how payment protection works"
✓ Keep CTAs active and direct: "Book a coach." not "Sessions can be booked."
✓ Use clay (#C4622D) only for active states, verified elements, and key accents
✓ Use JetBrains Mono exclusively for all currency values (₦)
Don't
✗ Never use tennis puns in serious contexts ("game, set, match for your career")
✗ Never stretch, skew, rotate, or recolour the logo mark
✗ Never use ball-yellow, court-blue, or clay-red as primary brand colours
✗ Never place the logo on a busy photographic background without a scrim
✗ Never over-Nigerianise copy to prove local credentials — it reads as patronising
✗ Never use passive voice in CTAs or marketing copy
